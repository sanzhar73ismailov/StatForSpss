# StatForSpss
project for prepare spss syntax files

##
Format: `yy.mm.dd.idx`
`yy`  - year, 2 symbols
`mm`  - month, 2 symbols
`dd`  - day, 2 symbols
`idx` - sequence number, starts from 1
